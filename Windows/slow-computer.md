# Troubleshooting a Slow Computer (Windows)

## Symptoms
- Programs take a long time to open
- System freezes or lags
- High CPU or memory usage

## Possible Causes
- Too many startup programs
- High resource usage
- Outdated system
- Malware

## Steps to Resolve

### 1. Check Task Manager
- Press Ctrl + Shift + Esc
- Click "Processes"
- Identify programs using high CPU or Memory
- End unnecessary tasks

### 2. Disable Startup Programs
- Open Task Manager
- Go to "Startup" tab
- Disable non-essential programs

### 3. Check for Windows Updates
- Go to Settings → Windows Update
- Install updates

### 4. Restart Computer
- Reboot system

## Additional Notes
If issue persists, check for malware or upgrade hardware (RAM/SSD).
