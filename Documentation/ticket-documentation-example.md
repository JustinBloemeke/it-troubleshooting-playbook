# Example Help Desk Ticket Documentation

## Ticket Summary
User unable to print to network printer. Printer shows as "Offline."

## User Details
- Department: Finance
- Device: Windows 10 Laptop
- Location: Office

## Issue Description
User reports that documents are stuck in the print queue and the printer appears offline. Other users may or may not be affected.

## Troubleshooting Steps Taken

1. Verified printer power and network connection
2. Confirmed device was connected to correct network
3. Checked printer status in Devices and Printers
4. Cleared print queue
5. Restarted Print Spooler service
6. Set printer to "Online"
7. Tested print job

## Resolution
Printer was set to "Use Printer Offline." After disabling this setting and restarting the spooler service, printing functionality was restored.

## Follow-Up Actions
- Advised user to report if issue reoccurs
- Documented resolution for future reference

## Notes
If issue persists across multiple users, escalate to network or print server support.
