# IT Support Knowledge Base

This repository contains practical troubleshooting guides for common IT support issues. These guides are designed to simulate real-world help desk scenarios and provide clear, step-by-step solutions for resolving technical problems.

## Topics Covered
- Windows troubleshooting
- Network connectivity issues
- Hardware diagnostics
- Printer support

## Purpose
This project demonstrates my ability to:
- Diagnose technical issues
- Document solutions clearly
- Communicate technical steps in a user-friendly way

## Tools & Concepts Used
- Windows troubleshooting tools (Task Manager, Device Manager, CMD)
- Basic networking commands (ipconfig, ping)
- Hardware diagnostics
- Step-by-step documentation
